- **회원가입**
- [ ] : ID 입력
- [ ] : ID 중복 확인
- [ ] : 닉네임 입력
- [ ] : Email 입력
- [ ] : 인증번호 발송
- [ ] : 인증번호 입력
- [ ] : 비밀번호 입력

- [ ] : ID 입력
- [ ] : PW 입력
- [ ] : 로그인 버튼 클릭
- **사용자 페이지**
- [ ] : 닉네임 수정
- [ ] : 내가 작성한 게시글 보기
- [ ] : 내가 작성한 댓글 보기

- [ ] : 게시글 리스트 (페이징적용)

- ** 상세 게시글 보기**
- [ ] : 제목 표시
- [ ] : 본문 표시

- [ ] : 생성 날짜 표시
- [ ] : 작성자 표시
- [ ] : 댓글 표시
- ** 게시글 작성**
- [ ] : 제목 입력
- [ ] : 본문 입력
- [ ] : 작성 버튼 클릭
- **게시글 수정**
- [ ] : 입력된 제목 수정
- [ ] : 입력된 본문 수정
- [ ] : 수정 버튼 클릭
- **게시글 삭제**
- [ ] : 삭제할것인지에 대한 Confirm 표시
- **댓글**
- [ ] : 댓글 리스트 (페이징 적용)
